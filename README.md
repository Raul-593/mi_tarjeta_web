### Ejercicios Prácticos para NIVEL 1 - Principiante - Frontend Responsivo
---
- ## Objetivo

Este proyecto muestra una tarjeta personal simple donde se presenta una breve descripción profesional. Incluye un botón que permite cambiar el color de fondo de la página de manera aleatoria, con el fin de agregar dinamismo y practicar conceptos básicos de manipulación del DOM.

- ## Tecnologías usadas

- HTML5 – Estructura del contenido
- CSS3 – Estilos y diseño de la tarjeta
- JavaScript – Cambio dinámico del color de fondo y manejo del DOM

- ## Cómo ejecutarlo

1. Descarga o clona este repositorio.
2. Abre el archivo index.html en tu navegador preferido.
3. Presiona el botón “Cambiar Color” para generar un color de fondo aleatorio.
